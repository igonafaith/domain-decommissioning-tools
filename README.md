# Domain Decommissioning Tools

## Latar Belakang

Alat ini ditujuankan untuk Full Stack Developer yang merangkap sebagai SEO/GEO Worker yang setiap harinya bekerja dengan puluhan atau ratusan domain-domain untuk keperluannya. Seperti yang kita tahu, sekali domain-domain ini dibeli, maka akan meninggalkan residu berupa fingerprint, footprint, bekas crawling dan banyak hal lainnya sehingga baik racun maupun reputasi dari brand anda akan tertinggal di domain tersebut meskipun domain itu sendiri kalian pakai sebagai penengah/proxy sehingga ketika dipakai lagi, informasi mengenai brand masih tertinggal dan berpotensi untuk dijadikan phishing. Untuk itu, tools ini berguna bukan untuk menghapus/membersihkan melainkan menurunkan potensi supaya domain-domain yang bekas dipakai dan tidak ingin dipakai/diperpanjang lagi untuk setidaknya secara nilai, domain ini memiliki nilai yang terdevaluasi sehingga jika dipakai untuk teknik negatif, mereka perlu bangun ulang kembali.

Alat ini menggabungkan beberapa alat keseharian yang biasa digunakan seperti Search Console atau Cloudflare dan menggunakan beberapa metode untuk membantu _decommissioning_ ini.
